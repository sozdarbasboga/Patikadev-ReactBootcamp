HTML Paragraphs

The HTML <p> element defines a paragraph.

This is a paragraph.

This is another paragraph.

The HTML <br> element inserts a single line break.

Bu satır kestirmesi olan
bir paragraftır.

The HTML <hr> element defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

        Yıldızlarla süslenmiş bir gökyüzü

        Ayın ışığıyla aydınlanıyor

        Kokusuyla sarhoş olan çiçekler,
        
        Doğanın kollarında huzur buluyor.
